What is method summary?

Method summary is a high level abstraction about what the method does. It doesn't include details.

Game Description

This game has only 8 levels as following:

  • 1-Starting to see the light
  • 2-Taking your first steps
  • 3-Good job! Keep on.
  • 4-Middle of the way
  • 5-Code squasher
  • Point worm
  • Proficient code summarizer
  • Monster slayer

But it doesn't mean that if you reach to level 8 you are winner. This game is based on the scores which you and other players will receive.

Top 10 players in Global Leaderboard and top 3 in Local Leaderboard are winners.

Local Leaderboard are defined based on the programming proficiency that players have.
