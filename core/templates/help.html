Help

What is method summary?

Method summary is a high level abstraction about what the method does. It doesn't include details.

Game Description

This game has only 8 levels as following:

  1. Starting to see the light
  2. Taking your first steps
  3. Good job! Keep on.
  4. Middle of the way
  5. Code squasher
  6. Point worm
  7. Proficient code summarizer
  8. Monster slayer

But it doesn't mean that if you reach to level 8 you are winner. This game is based on the scores which you and other players will receive.

Top 10 players in Global Leaderboard and top 3 in Local Leaderboard are winners.

Local Leaderboard are defined based on the programming proficiency that players have.

Game Menus

Summarization

You can see random snippets or select methods directly. Each method has a score(5 or 10 depends on the method difficulty). You can also see the number of comments on each method.

Hint: Your summaries will be evaluated by others and may decrease your points.

Evaluation

You can evaluate other comments by give a positive or negative point.

Hint: Your evaluations are captured.

Leaderboard

You can see top 10 and top 3 in global and local leaderboard.

Submit code

This menu will be activated for you and you can submit your own code if you reach to level 4. If it is approved, you can see the method in the game.

You can collect badges by submitting codes.

Badges

There are 18 badges that you can get along the game. By the end of the game based on your badges, you will be awarded.

Mystery box

You will receive random mystry box in different levels with random gifts such as 10xp, 0.25 extra mark, badge and empty.
